# Баги

- undefined вставляется при вставке символа, если текст изначально пустой
- при изменении угла крайней метки через инспектор, интерполяция не реагирует. также видимо угол группы не меняется
- при постоянном угле метки радиальный хелпер выделения неправильный
- при сдвиге крайнего деления якори не должны менять абсолютное положение
- в примере Регулятор при изменении угла крайней метки мышкой пляшет метка №6 (глюк компенсационных углов). Метка убегает от указателя мышки (при включенной компенсации угла)
- иногда после удаления объектов в списке остается какой-нибудь объект. пока не удалось понять, от чего это зависит.
- нужно обрабатывать недопустимые значения для атрибутов. Например, негативный размер прямоугольника.
- IE тестировать

# Доработки в текущей версии

- открытие файла по параметру адресной строки
- вместо "Абсолютный угол" сделать режимы: К центру(по умолч.)/Горизонталь(Нужно для угловых шкал)/Биссектриса (то что сейчас называется "Абсолютный угол")
- авто обрезка при экспорте в растровое изображение. Отключение через параметры конфигурации.
- радиус круга и отверстия аналогично арке редактировать мышью, и маркер центральный
