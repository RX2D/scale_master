# Баги

- IE тестировать!
- после доработки обрезки краев, проверить печать PNG, сравнить svg до и после экспорта
- отдельная метка, если ее повернуть и передвинуть - вспомогательные линии осей остаются на месте. Должны следовать за меткой.
- если метка была перемещена сдвигом, при перетягивании по углу она убегает от курсора мышки
- невозможно перетягиванием установить угол больше 360 град. Если при этом есть якори - деления сбиваются при переходе полного круга.
- при сдвиге мышкой крайнего деления якори меняют положение. этого не должно быть.
- изредка вместо углового сдвига деления включается радиальный сдвиг группы, с неправильным определенеем позиции указателя
- иногда после удаления объектов в списке остается какой-нибудь объект. пока не удалось понять, от чего это зависит.
- нужно обрабатывать недопустимые значения для атрибутов. Например, негативный размер прямоугольника.

# Доработки в текущей версии

- открытие файла по параметру адресной строки
- радиус круга и отверстия аналогично арке редактировать мышью, добавить центральный маркер
- маркеры для простых линий

# Новая версия

- интегрировать значение экспоненты в линейную интерполяцию. Тогда деления, распределенные по экспоненте, не будут сбрасываться в равномерное распределение при попытке их ручной корректировки.
- помечать заякоренные деления и метки при выборе групп
- менять вид курсора в разных режимах перетаскивания - учитывать Shift, Ctrl
- порядок объектов поменять на обратный порядку в файле. чтобы отображать объекты как слои в графическом редакторе.
- рамка шкалы - прямоугольник, ограниченный кругом 
- рамка шкалы - сегмент
- поле документа "автор"
- страница "О программе"
- сообщение о несовместимости для старых браузеров
- кнопка инвертирования цвета изображения (нужно ли?)
- заливка для замкнутых фигур и fill-opacity. поля ввода цвета должны допускать пустые значения - связанная доработка. Скорее всего, нужно использовать плагин ColorPicker.
